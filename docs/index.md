# libft

libft is a C library that reimplements commonly used functions from the C standard library and adds utility functions for strings, memory, output, and linked lists.

## Documentation

- Follow the [tutorial](tutorial/build-your-first-program.md) to build libft and use it in a small program.
- Read the [how-to guide](how-to/use-libft-in-another-project.md) to integrate libft into another C project.
- Use the [function reference](reference/functions.md) to look up available functions.
